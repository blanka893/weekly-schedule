# Weekly Schedule (Vite + React)

## Local run
```bash
npm install
npm run dev
```

## Deploy (Vercel)
- Import this repo in Vercel
- Build command: `npm run build`
- Output directory: `dist`

## Deploy (Netlify)
- Build command: `npm run build`
- Publish directory: `dist`
- (Optional SPA fix) Create `_redirects` with: `/* /index.html 200`
